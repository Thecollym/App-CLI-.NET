# App-CLI-.NET

La aplicación CLI fue creada por Luis R. Collymoore como un proyecto de práctica para familiarizarse con la programación en C# y la interacción con el sistema de administración de paquetes de Windows. Se desarrolló como una forma de aplicar y poner en práctica los conocimientos adquiridos en el aprendizaje de C# y la creación de aplicaciones de línea de comandos.

El objetivo principal de esta aplicación es proporcionar una experiencia interactiva para buscar, instalar, desinstalar y actualizar paquetes de software en un entorno de línea de comandos. Se ha diseñado de manera simple y eficiente, siguiendo las mejores prácticas de programación en C#.
